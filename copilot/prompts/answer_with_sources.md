# Resposta fundamentada

Responda à pergunta usando exclusivamente os trechos recuperados das fontes autorizadas.

Formato:

1. **Resposta:** síntese direta.
2. **Evidências:** bullets no formato `[Título — página/seção]`.
3. **Limitações:** lacunas, conflito ou desatualização encontrados.

Se não houver evidência suficiente, não tente completar com conhecimento externo. Solicite contexto ou encaminhe para revisão humana.
